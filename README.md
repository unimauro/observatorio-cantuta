# Observatorio La Cantuta

Portal ciudadano de transparencia de la **Universidad Nacional de Educación Enrique Guzmán y Valle** (UNE "La Cantuta", pliego 528). Solo datos públicos.

**Live:** https://unimauro.github.io/observatorio-cantuta/

Módulos: presupuesto y ejecución (MEF/SIAF), en qué se gasta cada sol, proveedores y dueños (OECE/CONOSCE), investigación (OpenAlex), gasto↔valor, personal (Portal de Transparencia). Nunca nombres de estudiantes.

Red de observatorios: [UNI](https://unimauro.github.io/observatorio-uni/) · [San Marcos](https://unimauro.github.io/observatorio-sanmarcos/) · [UNSAAC](https://unimauro.github.io/observatorio-unsaac/) · [Villarreal](https://unimauro.github.io/observatorio-villarreal/) · [UNSA](https://unimauro.github.io/observatorio-unsa/) · [ENSAD](https://unimauro.github.io/observatorio-ensad/). No es un portal oficial. Correcciones: carlos@cardenas.pe
